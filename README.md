# StudentHub

StudentHub is a personal dashboard for managing academics, projects, tasks, goals, and other parts of student life from a single platform.

The goal is to build something that feels less like a traditional productivity app and more like a personal command center. A place to keep track of coursework, coding projects, progress, and day-to-day tasks without everything being spread across different apps, folders, and browser tabs.

As the project grows, I plan to keep adding features that I personally find useful, whether that's tracking assignments, organizing notes, monitoring project progress, visualizing goals, or simply making it easier to stay on top of everything happening throughout the semester.

Some of the design inspiration comes from anime, games, and progression systems. I like the idea of a dashboard that doesn't just store information, but also makes progress feel visible and rewarding. The aim is to create something that is both practical and enjoyable to use.

StudentHub is also a space for me to experiment with web development, try out new ideas, and gradually build a tool that evolves alongside my needs as a student.
